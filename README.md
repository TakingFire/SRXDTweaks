# SRXD Tweaks

A collection of small plugins for Spin Rhythm XD. \
To download, see the [releases](https://github.com/TakingFire/SRXDTweaks/releases/latest) page.

## More Track Speeds

Doubles the in-menu track speed limit to 120.

## Touchable Medals

Adds cursor interaction to track medals.
